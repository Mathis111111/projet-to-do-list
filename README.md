Le but de ce projet et de creer une liste qu'on peut manipuler en ajoutant, en supprimant et en affichant des éléments.
Tout d'abord il est important de compiler pour que le programme puisse se lancer. pour cela il suffit d'éxécuter la commande "make" dans le terminal.
Ensuite il faut éxecuter le programme grâce à la commande "./organized" attention à ne pas entrer plus d'argument, cela vous renverai une sortie d'erreur.
Une fois cela fait, on peut commencer à manipuler le programme grâce à des commandes.
- "add" vous permet d'ajouter des éléments, il est suivi de 2 arguments, le premier est le type de tâche (mission, projet, rendez-vous, requête, ect...) et le deuxième et le descriptif de la tâche.
  ex : "add projet musculation"
- "del" vous permet de supprimer des éléments d'une liste, il est suivi uniquement du numéro de la tâche.
  ex : "del 1"
- "disp" vous permet d'afficher les éléments d'une liste, il est suivi d'aucun autre arguments.
  ex : "disp"
